Веб-сайт для ведения личного списка просмотренных фильмов или тех, что только планируются к просмотру. Проект демонстрирует работу с динамическими данными и локальным хранилищем браузера.

🚀 Основные возможности:

Добавление данных: Удобная форма для ввода названия, жанра, года выпуска и оценки.
Интерактивная оценка: Использование ползунка (range) с отображением текущего значения в реальном времени.
Локальное хранение (LocalStorage): Все добавленные фильмы сохраняются в памяти браузера. Список не пропадает даже после перезагрузки страницы.
Управление статусом: Возможность переключать статус «Успели посмотреть?» прямо в таблице (Да/Нет).
Удаление записей: Функция удаления фильма из списка и из памяти устройства при наведении на строку.

🛠 Технологии:

HTML5: Табличная верстка (table), формы и современные типы ввода.

CSS3: Стилизация интерфейса.

Эффекты при наведении (hover) для скрытия/появления кнопок удаления.

JavaScript (Vanilla JS): Обработка событий формы (submit, input).
Работа с массивами данных (map, filter, forEach).

LocalStorage API: Чтение и запись данных в формате JSON.
