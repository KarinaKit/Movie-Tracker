Стильное веб-приложение для ведения личной медиатеки. Позволяет не только хранить список любимых фильмов, но и гибко управлять ими, оценивать и быстро находить нужную информацию. Проект выполнен с использованием современного дизайна Glassmorphism (эффект матового стекла).

🌟 Основные функции:

Управление списком (CRUD): 
Добавление новых фильмов, редактирование данных и удаление ненужных записей.


Продвинутый поиск: 
«Живой» фильтр, который ищет совпадения одновременно по названию, жанру и году выпуска.


Интерактивный статус:
Возможность мгновенно переключить статус «Успели посмотреть?» (Да/Нет) прямо в строке таблицы.


Сохранение данных: 
Использование LocalStorage API гарантирует, что ваш список не обнулится после закрытия вкладки или обновления страницы.


Безопасность (Sanitization): 
Весь пользовательский ввод проходит очистку перед выводом на экран через textContent, что исключает риск XSS-атак.


Адаптивность: 
Интерфейс полностью оптимизирован под мобильные экраны (от 320px) с использованием медиа-запросов и горизонтального скролла для таблиц.


🛠 Технологии:
HTML5: 
Семантическая разметка и табличная структура данных.

CSS3:
Glassmorphism UI: полупрозрачные фоны и эффект backdrop-filter.

Responsive Design: 
гибкая сетка и адаптивные элементы.

JavaScript (ES6+):
Работа с асинхронным обновлением DOM.

Сложная фильтрация массивов (filter, includes).

Работа с локальным хранилищем браузера.
